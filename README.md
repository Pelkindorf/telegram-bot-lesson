# Run_bot — беговой Telegram-бот

Бот для учёта пробежек и статистики в Telegram.  
Написан на **Node.js** с использованием **Telegraf**.

## Возможности

- Добавление пробежек (дистанция, время, пульс, тип тренировки).
- Подсчёт объёма за день и неделю.
- Цель по километражу на неделю.
- Напоминания, если давно не было пробежек.
- Экспорт пробежек в CSV-файл.

## Запуск

1. Установить зависимости:

   ```bash
   npm install
   ```
